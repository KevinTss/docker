# Docker

Learning sandbox for [Docker](https://www.docker.com/)

## Run any docker image

```
docker run node
```

#### Parameters

| flag | description |
| --- | --- |
| -it | Stay with the terminal open (if not, he will launch the command in the background |

---

Check all containers
```
docker ps
```

| flag | description |
| --- | --- |
| -a | see also stopped containers |
